## Build your website in minutes

Beautiful Jekyll is a ready-to-use template to help you create a beautiful website quickly. Perfect for personal sites, blogs, or simple project websites. :contentReference[oaicite:1]{index=1}

---

<div class="gs-section-01" markdown="1">

### 1. Fork the Beautiful Jekyll repository

Fork the [Beautiful Jekyll GitHub repo](https://github.com/daattali/beautiful-jekyll) by clicking the **Fork** button in the top right. This makes a copy of the template in your GitHub account.

</div>

---

<div class="gs-section-02" markdown="1">

### 2. Rename your repository

After forking, rename your repository to `<yourusername>.github.io` in GitHub **Settings → Repository name**. This tells GitHub to publish it as a user site.

</div>

---

<div class="gs-section-03" markdown="1">

### 3. Customize your config file

Edit `_config.yml` to update your site’s name, colors, and navigation links. Commit your changes.

</div>
